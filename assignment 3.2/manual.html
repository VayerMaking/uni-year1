Bridge

Rules

The main goal of the game is guessing words. The game is educational, makes you learn new words and develops thinking.

The player sees how many letters is the word then enters a letter. If the letter is correct it is displayed and if is not one of the pillars of the bridge falls.

If all the pillars fall it's game over.
